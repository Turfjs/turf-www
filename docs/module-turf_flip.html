turf.flip

Takes a point, linestring, polygon, or featurecollection, and flips all of its coordinates from [x, y] to [y, x].

Parameters

Name Type Description
input *

Returns

FeatureCollection

output

Example

var poly = turf.polygon([[[1,0], [1,0], [1,2]], [[.2,.2], [.3,.3],[.1,.2]]])
var flipped = turf.flip(poly)
console.log(flipped)
