### Description

Takes any number of features and returns a rectangular [Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6) that encompasses all vertices.

### Parameters

| Name    | Type                                                         | Description    |
| ------- | ------------------------------------------------------------ | -------------- |
| geojson | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** | input features |

### Returns

**[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)&lt;[Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6)&gt;** a rectangular Polygon feature that encompasses all vertices

### Examples

```javascript
var features = turf.featureCollection([
  turf.point([-75.343, 39.984], { name: "Location A" }),
  turf.point([-75.833, 39.284], { name: "Location B" }),
  turf.point([-75.534, 39.123], { name: "Location C" }),
]);

var enveloped = turf.envelope(features);
```
