### Description

Takes a bbox and returns an equivalent [polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6).

### Parameters

| Name                       | Type                                                                                                                                                                                               | Description                                              |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| bbox                       | **[BBox](https://tools.ietf.org/html/rfc7946#section-5)**                                                                                                                                          | extent in [minX, minY, maxX, maxY] order                 |
| options<i>?</i>            | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)**                                                                                                    | Optional parameters _(default &#123;&#125;)_             |
| options.properties<i>?</i> | **[GeoJsonProperties](https://tools.ietf.org/html/rfc7946#section-3.2)**                                                                                                                           | Translate properties to Polygon _(default &#123;&#125;)_ |
| options.id<i>?</i>         | **[string](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/String) &#124; [number](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number)** | Translate Id to Polygon _(default &#123;&#125;)_         |

### Returns

**[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)&lt;[Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6)&gt;** a Polygon representation of the bounding box

### Examples

```javascript
var bbox = [0, 0, 10, 10];

var poly = turf.bboxPolygon(bbox);
```
