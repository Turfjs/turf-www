### Description

Convert a distance measurement (assuming a spherical Earth) from a real-world unit into radians
Valid units: miles, nauticalmiles, inches, yards, meters, metres, kilometers, centimeters, feet

### Parameters

| Name          | Type                                                                                            | Description                                                                                                     |
| ------------- | ----------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| distance      | **[number](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number)** | in real units                                                                                                   |
| units<i>?</i> | **[Units](docs/api/types/Units.mdx)**                                                           | can be degrees, radians, miles, inches, yards, metres, meters, kilometres, kilometers. _(default "kilometers")_ |

### Returns

**[number](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number)** radians

### Installation

```javascript
$ npm install @turf/helpers

import { lengthToRadians } from "@turf/helpers";
const result = lengthToRadians(...);
```

```javascript
$ npm install @turf/turf

import * as turf from "@turf/turf";
const result = turf.lengthToRadians(...);
```
