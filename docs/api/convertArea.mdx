### Description

Converts a area to the requested unit.
Valid units: kilometers, kilometres, meters, metres, centimetres, millimeters, acres, miles, yards, feet, inches, hectares

### Parameters

| Name                 | Type       | Description                            |
| -------------------- | ---------- | -------------------------------------- |
| area                 | **number** | to be converted                        |
| originalUnit<i>?</i> | **Units**  | of the distance _(default "meters")_   |
| finalUnit<i>?</i>    | **Units**  | returned unit _(default "kilometers")_ |

### Returns

<ul>
  **number** the converted area

</ul>

### Installation

```javascript
$ npm install @turf/helpers

import { convertArea } from "@turf/helpers";
const result = convertArea(...);
```

```javascript
$ npm install @turf/turf

import * as turf from "@turf/turf";
const result = turf.convertArea(...);
```
