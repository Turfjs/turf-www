### Description

Converts a Mercator (EPSG:900913) GeoJSON object into WGS84 projection

### Parameters

| Name                   | Type                                                                     | Description                                                                                     |
| ---------------------- | ------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| geojson                | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3) \| Position** | Mercator GeoJSON object                                                                         |
| options<i>?</i>        | **Object**                                                               | Optional parameters                                                                             |
| options.mutate<i>?</i> | **boolean**                                                              | allows GeoJSON input to be mutated (significant performance increase if true) _(default false)_ |

### Returns

<ul>
  **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** Projected GeoJSON

</ul>

### Examples

```javascript
var pt = turf.point([-7903683.846322424, 5012341.663847514]);
var converted = turf.toWgs84(pt);
```
