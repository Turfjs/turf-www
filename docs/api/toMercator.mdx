### Description

Converts a WGS84 GeoJSON object into Mercator (EPSG:900913) projection

### Parameters

| Name                   | Type                                                                     | Description                                                                                     |
| ---------------------- | ------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| geojson                | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3) \| Position** | WGS84 GeoJSON object                                                                            |
| options<i>?</i>        | **Object**                                                               | Optional parameters                                                                             |
| options.mutate<i>?</i> | **boolean**                                                              | allows GeoJSON input to be mutated (significant performance increase if true) _(default false)_ |

### Returns

<ul>
  **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** Projected GeoJSON

</ul>

### Examples

```javascript
var pt = turf.point([-71, 41]);
var converted = turf.toMercator(pt);
```
