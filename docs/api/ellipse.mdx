### Description

Takes a [Point](https://tools.ietf.org/html/rfc7946#section-3.1.2) and calculates the ellipse polygon given two semi-axes expressed in variable units and steps for precision.

### Parameters

| Name                       | Type                                                           | Description                                                            |
| -------------------------- | -------------------------------------------------------------- | ---------------------------------------------------------------------- |
| center                     | **[Coord](https://tools.ietf.org/html/rfc7946#section-3.1.1)** | center point                                                           |
| xSemiAxis                  | **number**                                                     | semi (major) axis of the ellipse along the x-axis                      |
| ySemiAxis                  | **number**                                                     | semi (minor) axis of the ellipse along the y-axis                      |
| options<i>?</i>            | **Object**                                                     | Optional parameters _(default \{})_                                    |
| options.angle<i>?</i>      | **number**                                                     | angle of rotation in decimal degrees, positive clockwise _(default 0)_ |
| options.pivot<i>?</i>      | **[Coord](https://tools.ietf.org/html/rfc7946#section-3.1.1)** | point around which the rotation will be performed _(default 'origin')_ |
| options.steps<i>?</i>      | **number**                                                     | number of steps _(default 64)_                                         |
| options.units<i>?</i>      | **string**                                                     | unit of measurement for axes _(default 'kilometers')_                  |
| options.properties<i>?</i> | **Object**                                                     | properties _(default \{})_                                             |

### Returns

<ul>
  **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6)>** ellipse polygon

</ul>

### Examples

```javascript
var center = [-75, 40];
var xSemiAxis = 5;
var ySemiAxis = 2;
var ellipse = turf.ellipse(center, xSemiAxis, ySemiAxis);
```
