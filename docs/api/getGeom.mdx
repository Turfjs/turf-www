### Description

Get Geometry from Feature or Geometry Object

### Parameters

| Name    | Type                                                                                                                          | Description                        |
| ------- | ----------------------------------------------------------------------------------------------------------------------------- | ---------------------------------- |
| geojson | **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2) \| [Geometry](https://tools.ietf.org/html/rfc7946#section-3.1)** | GeoJSON Feature or Geometry Object |

### Returns

**[Geometry](https://tools.ietf.org/html/rfc7946#section-3.1) \| null** GeoJSON Geometry Object

### Examples

```javascript
var point = {
  type: "Feature",
  properties: {},
  geometry: {
    type: "Point",
    coordinates: [110, 40],
  },
};
var geom = turf.getGeom(point);
//={"type": "Point", "coordinates": [110, 40]}
```

### Installation

```javascript
$ npm install @turf/invariant

import { getGeom } from "@turf/invariant";

```
