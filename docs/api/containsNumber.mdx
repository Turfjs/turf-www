### Description

Checks if coordinates contains a number

### Parameters

| Name        | Type                                                                                                     | Description         |
| ----------- | -------------------------------------------------------------------------------------------------------- | ------------------- |
| coordinates | **[Array](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Array)&lt;any&gt;** | GeoJSON Coordinates |

### Returns

**[boolean](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Boolean)** true if Array contains a number

### Installation

```javascript
$ npm install @turf/invariant

import { containsNumber } from "@turf/invariant";
const result = containsNumber(...);
```

```javascript
$ npm install @turf/turf

import * as turf from "@turf/turf";
const result = turf.containsNumber(...);
```
