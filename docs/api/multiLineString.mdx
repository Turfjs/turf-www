### Description

Creates a [Feature\<MultiLineString\>](Feature<MultiLineString>) based on a
coordinate array. Properties can be added optionally.

### Parameters

| Name                 | Type                                  | Description                                                               |
| -------------------- | ------------------------------------- | ------------------------------------------------------------------------- |
| coordinates          | **Array\<Array\<Array\<number\>\>\>** | an array of LineStrings                                                   |
| properties<i>?</i>   | **Object**                            | an Object of key-value pairs to add as properties _(default \{\})_        |
| options<i>?</i>      | **Object**                            | Optional Parameters _(default \{\})_                                      |
| options.bbox<i>?</i> | **Array\<number\>**                   | Bounding Box Array [west, south, east, north] associated with the Feature |
| options.id<i>?</i>   | **string \| number**                  | Identifier associated with the Feature                                    |

### Returns

<ul>
  **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[MultiLineString](https://tools.ietf.org/html/rfc7946#section-3.1.5)\>** a MultiLineString feature

</ul>

### Examples

```javascript
var multiLine = turf.multiLineString([
  [
    [0, 0],
    [10, 10],
  ],
]);

//=multiLine
```

### Installation

```javascript
$ npm install @turf/helpers

import { multiLineString } from "@turf/helpers";
const result = multiLineString(...);
```
