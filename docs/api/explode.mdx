### Description

Takes a feature or set of features and returns all positions as [points](https://tools.ietf.org/html/rfc7946#section-3.1.2).

### Parameters

| Name    | Type                                                         | Description    |
| ------- | ------------------------------------------------------------ | -------------- |
| geojson | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** | input features |

### Returns

<ul>
  **[FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3)\<point>** points representing the exploded input features

</ul>

### Examples

```javascript
var polygon = turf.polygon([
  [
    [-81, 41],
    [-88, 36],
    [-84, 31],
    [-80, 33],
    [-77, 39],
    [-81, 41],
  ],
]);

var explode = turf.explode(polygon);
```
