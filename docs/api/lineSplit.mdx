### Description

Split a LineString by another GeoJSON Feature.

### Parameters

| Name     | Type                                                                                                                                  | Description                 |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------- | --------------------------- |
| line     | **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)&lt;[LineString](https://tools.ietf.org/html/rfc7946#section-3.1.4)&gt;** | LineString Feature to split |
| splitter | **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)&lt;any&gt;**                                                             | Feature used to split line  |

### Returns

**[FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3)&lt;[LineString](https://tools.ietf.org/html/rfc7946#section-3.1.4)&gt;** Split LineStrings

### Examples

```javascript
var line = turf.lineString([
  [120, -25],
  [145, -25],
]);
var splitter = turf.lineString([
  [130, -15],
  [130, -35],
]);

var split = turf.lineSplit(line, splitter);
```
