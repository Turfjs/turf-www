### Description

Linear measurement units.

⚠️ Warning. Be aware of the implications of using radian or degree units to
measure distance. The distance represented by a degree of longitude _varies_
depending on latitude.

See [https://www.thoughtco.com/degree-of-latitude-and-longitude-distance-4070616](https://www.thoughtco.com/degree-of-latitude-and-longitude-distance-4070616)
for an illustration of this behaviour.
