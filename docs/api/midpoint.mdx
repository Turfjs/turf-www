### Description

Takes two points and returns a point midway between them. The midpoint is
calculated geodesically, meaning the curvature of the earth is taken into
account.

### Parameters

| Name   | Type                                                           | Description  |
| ------ | -------------------------------------------------------------- | ------------ |
| point1 | **[Coord](https://tools.ietf.org/html/rfc7946#section-3.1.1)** | first point  |
| point2 | **[Coord](https://tools.ietf.org/html/rfc7946#section-3.1.1)** | second point |

### Returns

**[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[Point](https://tools.ietf.org/html/rfc7946#section-3.1.2)\>** a point midway between `pt1` and `pt2`

### Examples

```javascript
const point1 = turf.point([144.834823, -37.771257]);
const point2 = turf.point([145.14244, -37.830937]);

const midpoint = turf.midpoint(point1, point2);
```

```javascript live
function Map() {

```
