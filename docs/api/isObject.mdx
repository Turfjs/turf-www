### Description

isObject

### Parameters

| Name  | Type   | Description          |
| ----- | ------ | -------------------- |
| input | **\*** | variable to validate |

### Returns

<ul>
  **boolean** true/false

</ul>

### Examples

```javascript
turf.isObject({ elevation: 10 });
//=true
turf.isObject("foo");
//=false
```
