### Description

Takes a [GeoJSON](https://tools.ietf.org/html/rfc7946#section-3) object and returns a simplified version. Internally uses
[simplify-js](http://mourner.github.io/simplify-js/) to perform simplification using the Ramer-Douglas-Peucker algorithm.

### Parameters

| Name                        | Type                                                         | Description                                                                                                              |
| --------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| geojson                     | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** | object to be simplified                                                                                                  |
| options<i>?</i>             | **Object**                                                   | Optional parameters _(default \{})_                                                                                      |
| options.tolerance<i>?</i>   | **number**                                                   | simplification tolerance _(default 1)_                                                                                   |
| options.highQuality<i>?</i> | **boolean**                                                  | whether or not to spend more time to create a higher-quality simplification with a different algorithm _(default false)_ |
| options.mutate<i>?</i>      | **boolean**                                                  | allows GeoJSON input to be mutated (significant performance increase if true) _(default false)_                          |

### Returns

<ul>
  **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** a simplified GeoJSON

</ul>
