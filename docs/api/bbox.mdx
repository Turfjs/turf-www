### Description

Calculates the bounding box for any GeoJSON object, including FeatureCollection.
Uses geojson.bbox if available and options.recompute is not set.

### Parameters

| Name                      | Type                                                                                              | Description                                            |
| ------------------------- | ------------------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| geojson                   | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)**                                      | any GeoJSON object                                     |
| options<i>?</i>           | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)**   | Optional parameters _(default \{\})_                   |
| options.recompute<i>?</i> | **[boolean](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Boolean)** | Whether to ignore an existing bbox property on geojson |

### Returns

**[BBox](https://tools.ietf.org/html/rfc7946#section-5)** bbox extent in [minX, minY, maxX, maxY] order

### Examples

```javascript
var line = turf.lineString([
  [-74, 40],
  [-78, 42],
  [-82, 35],
]);
var bbox = turf.bbox(line);
var bboxPolygon = turf.bboxPolygon(bbox);
```
