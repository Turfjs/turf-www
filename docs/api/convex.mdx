### Description

Takes a [Feature](https://tools.ietf.org/html/rfc7946#section-3.2) or a [FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3) and returns a convex hull [Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6).

Internally this uses
the [convex-hull](https://github.com/mikolalysenko/convex-hull) module that implements a
[monotone chain hull](http://en.wikibooks.org/wiki/Algorithm_Implementation/Geometry/Convex_hull/Monotone_chain).

### Parameters

| Name                       | Type                                                                                            | Description                                                  |
| -------------------------- | ----------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| geojson                    | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)**                                    | input Feature or FeatureCollection                           |
| options<i>?</i>            | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)** | Optional parameters _(default &#123;&#125;)_                 |
| options.concavity<i>?</i>  | **[number](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number)** | 1 - thin shape. Infinity - convex hull. _(default Infinity)_ |
| options.properties<i>?</i> | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)** | Translate Properties to Feature _(default &#123;&#125;)_     |

### Returns

**[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)&lt;[Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6)&gt;** a convex hull
