### Description

Computes the centroid as the mean of all vertices within the object.

### Parameters

| Name                       | Type                                                                                            | Description                                                                                                            |
| -------------------------- | ----------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| geojson                    | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)**                                    | GeoJSON to be centered                                                                                                 |
| options<i>?</i>            | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)** | Optional Parameters _(default \{\})_                                                                                   |
| options.properties<i>?</i> | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)** | an Object that is used as the [Feature](https://tools.ietf.org/html/rfc7946#section-3.2)'s properties _(default \{\})_ |

### Returns

**[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[Point](https://tools.ietf.org/html/rfc7946#section-3.1.2)\>** the centroid of the input object

### Examples

```javascript
var polygon = turf.polygon([
  [
    [-81, 41],
    [-88, 36],
    [-84, 31],
    [-80, 33],
    [-77, 39],
    [-81, 41],
  ],
]);

var centroid = turf.centroid(polygon);
```
