### Description

Takes one or more features and calculates the centroid using the mean of all vertices.
This lessens the effect of small islands and artifacts when calculating the centroid of a set of polygons.

### Parameters

| Name                       | Type                                                         | Description                                                                 |
| -------------------------- | ------------------------------------------------------------ | --------------------------------------------------------------------------- |
| geojson                    | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** | GeoJSON to be centered                                                      |
| options<i>?</i>            | **Object**                                                   | Optional Parameters _(default \{})_                                         |
| options.properties<i>?</i> | **Object**                                                   | an Object that is used as the \{@link Feature}'s properties _(default \{})_ |

### Returns

<ul>
  **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[Point](https://tools.ietf.org/html/rfc7946#section-3.1.2)>** the centroid of the input features

</ul>

### Examples

```javascript
var polygon = turf.polygon([
  [
    [-81, 41],
    [-88, 36],
    [-84, 31],
    [-80, 33],
    [-77, 39],
    [-81, 41],
  ],
]);

var centroid = turf.centroid(polygon);
```
