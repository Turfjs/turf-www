### Description

Creates a [Feature](https://tools.ietf.org/html/rfc7946#section-3.2)&lt;[MultiPoint](https://tools.ietf.org/html/rfc7946#section-3.1.3)&gt; based on a
coordinate array. Properties can be added optionally.

### Parameters

| Name                 | Type                                                                                                          | Description                                                                |
| -------------------- | ------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| coordinates          | **[Array](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Array)&lt;Position&gt;** | an array of Positions                                                      |
| properties<i>?</i>   | **[GeoJsonProperties](https://tools.ietf.org/html/rfc7946#section-3.2)**                                      | an Object of key-value pairs to add as properties _(default &#123;&#125;)_ |
| options<i>?</i>      | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)**               | Optional Parameters _(default &#123;&#125;)_                               |
| options.bbox<i>?</i> | **[BBox](https://tools.ietf.org/html/rfc7946#section-5)**                                                     | Bounding Box Array [west, south, east, north] associated with the Feature  |
| options.id<i>?</i>   | **Id**                                                                                                        | Identifier associated with the Feature                                     |

### Returns

**[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)&lt;[MultiPoint](https://tools.ietf.org/html/rfc7946#section-3.1.3), [GeoJsonProperties](https://tools.ietf.org/html/rfc7946#section-3.2)&gt;** a MultiPoint feature

### Examples

```javascript
var multiPt = turf.multiPoint([
  [0, 0],
  [10, 10],
]);

//=multiPt
```

### Installation

```javascript
$ npm install @turf/helpers

import { multiPoint } from "@turf/helpers";
const result = multiPoint(...);
```

```javascript
$ npm install @turf/turf

import * as turf from "@turf/turf";

```
