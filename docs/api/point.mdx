### Description

Creates a [Point](https://tools.ietf.org/html/rfc7946#section-3.1.2) [Feature](https://tools.ietf.org/html/rfc7946#section-3.2) from a Position.

### Parameters

| Name                 | Type                 | Description                                                               |
| -------------------- | -------------------- | ------------------------------------------------------------------------- |
| coordinates          | **Array\<number>**   | longitude, latitude position (each in decimal degrees)                    |
| properties<i>?</i>   | **Object**           | an Object of key-value pairs to add as properties _(default \{})_         |
| options<i>?</i>      | **Object**           | Optional Parameters _(default \{})_                                       |
| options.bbox<i>?</i> | **Array\<number>**   | Bounding Box Array [west, south, east, north] associated with the Feature |
| options.id<i>?</i>   | **string \| number** | Identifier associated with the Feature                                    |

### Returns

<ul>
  **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[Point](https://tools.ietf.org/html/rfc7946#section-3.1.2)>** a Point feature

</ul>

### Examples

```javascript
var point = turf.point([-75.343, 39.984]);

//=point
```

### Installation

```javascript
$ npm install @turf/helpers

import { point } from "@turf/helpers";
const result = point(...);
```

```javascript
$ npm install @turf/turf

import * as turf from "@turf/turf";
const result = turf.point(...);
```
