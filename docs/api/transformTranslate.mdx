### Description

Moves any geojson Feature or Geometry of a specified distance along a Rhumb Line
on the provided direction angle.

### Parameters

| Name                         | Type                                                         | Description                                                                                          |
| ---------------------------- | ------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| geojson                      | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** | object to be translated                                                                              |
| distance                     | **number**                                                   | length of the motion; negative values determine motion in opposite direction                         |
| direction                    | **number**                                                   | of the motion; angle from North in decimal degrees, positive clockwise                               |
| options<i>?</i>              | **Object**                                                   | Optional parameters _(default \{})_                                                                  |
| options.units<i>?</i>        | **string**                                                   | in which `distance` will be express; miles, kilometers, degrees, or radians _(default 'kilometers')_ |
| options.zTranslation<i>?</i> | **number**                                                   | length of the vertical motion, same unit of distance _(default 0)_                                   |
| options.mutate<i>?</i>       | **boolean**                                                  | allows GeoJSON input to be mutated (significant performance increase if true) _(default false)_      |

### Returns

<ul>
  **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** the translated GeoJSON object

</ul>

### Examples

```javascript
var poly = turf.polygon([
  [
    [0, 29],
    [3.5, 29],
    [2.5, 32],
    [0, 29],
  ],
]);
var translatedPoly = turf.transformTranslate(poly, 100, 35);
```
