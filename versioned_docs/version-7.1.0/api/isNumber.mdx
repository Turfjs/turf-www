### Description

isNumber

### Parameters

| Name | Type   | Description        |
| ---- | ------ | ------------------ |
| num  | **\*** | Number to validate |

### Returns

<ul>
  **boolean** true/false

</ul>

### Examples

```javascript
turf.isNumber(123);
//=true
turf.isNumber("foo");
//=false
```
