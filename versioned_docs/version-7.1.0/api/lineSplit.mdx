### Description

Split a LineString by another GeoJSON Feature.

### Parameters

| Name     | Type                                                                                                                              | Description                 |
| -------- | --------------------------------------------------------------------------------------------------------------------------------- | --------------------------- |
| line     | **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[LineString](https://tools.ietf.org/html/rfc7946#section-3.1.4)\>** | LineString Feature to split |
| splitter | **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<any\>**                                                             | Feature used to split line  |

### Returns

<ul>
  **[FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3)\<[LineString](https://tools.ietf.org/html/rfc7946#section-3.1.4)\>** Split LineStrings

</ul>

### Examples

```javascript
var line = turf.lineString([
  [120, -25],
  [145, -25],
]);
var splitter = turf.lineString([
  [130, -15],
  [130, -35],
]);

var split = turf.lineSplit(line, splitter);
```
