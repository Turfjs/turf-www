### Description

Creates a [FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3) of 2-vertex [LineString](https://tools.ietf.org/html/rfc7946#section-3.1.4) segments from a
[(Multi)LineString](https://tools.ietf.org/html/rfc7946#section-3.1.4) or [(Multi)Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6).

### Parameters

| Name    | Type                                                         | Description                   |
| ------- | ------------------------------------------------------------ | ----------------------------- |
| geojson | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)** | GeoJSON Polygon or LineString |

### Returns

**[FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3)\<[LineString](https://tools.ietf.org/html/rfc7946#section-3.1.4)\>** 2-vertex line segments

### Examples

```javascript
var polygon = turf.polygon([
  [
    [-50, 5],
    [-40, -10],
    [-50, -10],
    [-40, 5],
    [-50, 5],
  ],
]);
var segments = turf.lineSegment(polygon);
```
