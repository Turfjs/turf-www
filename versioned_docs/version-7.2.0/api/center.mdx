### Description

Takes a [Feature](https://tools.ietf.org/html/rfc7946#section-3.2) or [FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3) and returns the absolute center point of all features.

### Parameters

| Name                       | Type                                                                                            | Description                                            |
| -------------------------- | ----------------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| geojson                    | **[GeoJSON](https://tools.ietf.org/html/rfc7946#section-3)**                                    | GeoJSON to be centered                                 |
| options<i>?</i>            | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)** | Optional parameters _(default \{\})_                   |
| options.properties<i>?</i> | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)** | Translate GeoJSON Properties to Point _(default \{\})_ |
| options.bbox<i>?</i>       | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)** | Translate GeoJSON BBox to Point _(default \{\})_       |
| options.id<i>?</i>         | **[Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)** | Translate GeoJSON Id to Point _(default \{\})_         |

### Returns

**[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[Point](https://tools.ietf.org/html/rfc7946#section-3.1.2)\>** a Point feature at the absolute center point of all input features

### Examples

```javascript
var features = turf.points([
  [-97.522259, 35.4691],
  [-97.502754, 35.463455],
  [-97.508269, 35.463245],
]);

var center = turf.center(features);
```

```javascript live
function Map() {
  var features = turf.points([
    [-97.522259, 35.4691],
    [-97.502754, 35.463455],
    [-97.508269, 35.463245],
  ]);

  var center = turf.center(features);
  center.properties["marker-size"] = "large";
  center.properties["marker-color"] = "#000";

  return <ExampleMap addToMap={{ features, center }} />;
}
```
