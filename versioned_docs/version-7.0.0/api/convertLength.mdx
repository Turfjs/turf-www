### Description

Converts a length to the requested unit.
Valid units: miles, nauticalmiles, inches, yards, meters, metres, kilometers, centimeters, feet

### Parameters

| Name                 | Type       | Description                            |
| -------------------- | ---------- | -------------------------------------- |
| length               | **number** | to be converted                        |
| originalUnit<i>?</i> | **Units**  | of the length _(default "kilometers")_ |
| finalUnit<i>?</i>    | **Units**  | returned unit _(default "kilometers")_ |

### Returns

<ul>
  **number** the converted length

</ul>

### Installation

```javascript
$ npm install @turf/helpers

import { convertLength } from "@turf/helpers";
const result = convertLength(...);
```

```javascript
$ npm install @turf/turf

import * as turf from "@turf/turf";
const result = turf.convertLength(...);
```
