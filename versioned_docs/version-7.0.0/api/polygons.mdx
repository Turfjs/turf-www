### Description

Creates a [Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6) [FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3) from an Array of Polygon coordinates.

### Parameters

| Name                 | Type                                           | Description                                                               |
| -------------------- | ---------------------------------------------- | ------------------------------------------------------------------------- |
| coordinates          | **Array\<Array\<Array\<Array\<number\>\>\>\>** | an array of Polygon coordinates                                           |
| properties<i>?</i>   | **Object**                                     | an Object of key-value pairs to add as properties _(default \{\})_        |
| options<i>?</i>      | **Object**                                     | Optional Parameters _(default \{\})_                                      |
| options.bbox<i>?</i> | **Array\<number\>**                            | Bounding Box Array [west, south, east, north] associated with the Feature |
| options.id<i>?</i>   | **string \| number**                           | Identifier associated with the FeatureCollection                          |

### Returns

<ul>
  **[FeatureCollection](https://tools.ietf.org/html/rfc7946#section-3.3)\<[Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6)\>** Polygon FeatureCollection

</ul>

### Examples

```javascript
var polygons = turf.polygons([
  [
    [
      [-5, 52],
      [-4, 56],
      [-2, 51],
      [-7, 54],
      [-5, 52],
    ],
  ],
  [
    [
      [-15, 42],
      [-14, 46],
      [-12, 41],
      [-17, 44],
      [-15, 42],
    ],
  ],
]);

//=polygons
```

### Installation

```javascript
$ npm install @turf/helpers

import { polygons } from "@turf/helpers";
const result = polygons(...);
```

```javascript
$ npm install @turf/turf

import * as turf from "@turf/turf";

```
