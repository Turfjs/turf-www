### Description

Creates a [Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6) [Feature](https://tools.ietf.org/html/rfc7946#section-3.2) from an Array of LinearRings.

### Parameters

| Name                 | Type                                  | Description                                                               |
| -------------------- | ------------------------------------- | ------------------------------------------------------------------------- |
| coordinates          | **Array\<Array\<Array\<number\>\>\>** | an array of LinearRings                                                   |
| properties<i>?</i>   | **Object**                            | an Object of key-value pairs to add as properties _(default \{\})_        |
| options<i>?</i>      | **Object**                            | Optional Parameters _(default \{\})_                                      |
| options.bbox<i>?</i> | **Array\<number\>**                   | Bounding Box Array [west, south, east, north] associated with the Feature |
| options.id<i>?</i>   | **string \| number**                  | Identifier associated with the Feature                                    |

### Returns

<ul>
  **[Feature](https://tools.ietf.org/html/rfc7946#section-3.2)\<[Polygon](https://tools.ietf.org/html/rfc7946#section-3.1.6)\>** Polygon Feature

</ul>

### Examples

```javascript
var polygon = turf.polygon(
  [
    [
      [-5, 52],
      [-4, 56],
      [-2, 51],
      [-7, 54],
      [-5, 52],
    ],
  ],
  { name: "poly1" },
);

//=polygon
```

### Installation

```javascript
$ npm install @turf/helpers

import { polygon } from "@turf/helpers";
const result = polygon(...);
```
